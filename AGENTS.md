# Agent Instructions

This repository is a beginner-friendly ROS 2 tutorial course.

Before making changes, agents must read and follow `CLAUDE.md`. Treat `CLAUDE.md` as the main teaching, writing, and lesson-generation contract for this project.

Preserve these priorities in all edits:

- Teach ROS 2 fundamentals before advanced robotics tools.
- Keep the material practical, hands-on, and beginner-friendly.
- Assume the learner is new to ROS 2.
- Prefer lightweight, low-storage workflows early in the course.
- Avoid introducing Gazebo, Navigation2, MoveIt, Docker, YOLO, AI packages, large simulation worlds, or full desktop robotics stacks in beginner lessons unless clearly marked as future work.
- Write like a patient expert teacher who explains both what to do and why it matters.

The main roadmap is `Lessons/README.md`.

Generated lesson plans should go in `Lessons/lesson-plan/`.

Generated student-facing lessons should go in `Lessons/lessons/`.
